Para rodar o programa pcv.py, siga o seguintes passos: 

1 - Instale o Python no seu computador;
2 - No terminal, acesse o diretório do arquivo selection.py (__dirname + '/APA/Projeto Final/E2_P2');
3 (Opcional) - Edite o arquivo de entrada do programa (bayg29.txt);
4 - Execute o comando '$ python pcv.py'
5 - Digite o nome do arquivo txt

----------------------------------------------

Para essa etapa do projeto foi utilizado a metaheurística do Simulated Annealing. Como construção inicial,
o algoritmo do vizinho mais próximo foi usado. Os resultados apresentados do Simulated Annealing sempre
mostraram uma distancia final maior do que a da heurística de construção e os valores nunca mudavam,
logo isso explica o porquê do GAP sempre apresentar um valor 0.
